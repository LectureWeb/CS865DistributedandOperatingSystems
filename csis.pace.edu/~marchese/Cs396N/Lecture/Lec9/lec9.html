Sending Data to Server via HTML Forms: Get & Post

  • Create Interface controls to collect input on Web page
  • Each contol has name and value
    • Name in HTML
    • Value from HTML or user input
  • Entire form associated with URL of program that processed the data
  • Upon data submission, names and values sent to designated URL as string
  • String Sent as Get or Post
    • HTTP GET - appends string to end of specified URL
    • HTTP POST - HTTP request header and blank line sent then data on following line

Sending Data with GET
